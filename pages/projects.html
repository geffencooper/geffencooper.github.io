Facial Temperature Scanner

UCSB ECE 153B Project
Winter Quarter 2021

Project Website
Embedded Deep Learning PyTorch C, Python

  • For my ECE 153B (Sensor and Peripheral Interface Design) project, I built a facial temperature scanner using the MAX78000FTHR development board that reports if a person has a safe temperature.

  • The scanner uses the on-board camera module and CNN accelerator to detect a person’s face in the image frame. The camera feed and instructions (‘move left’, ‘move up’, etc.) are displayed on an LCD to help the person center their face..

  • Once centered, an IR temperature sensor will measure the person’s facial temperature which is then reported on the LCD

  • Further details can be found on the project website above.

Personal Project

GitHub Repo

Tracking Robot

Personal Project
November 2020 - January 2021

GitHub Repo
Embedded OpenCV, ROS C, C++, Python

  • I built a robot using an RC car, STM32, and Raspberry Pi that can track a person using ROS and OpenCV.

  • The STM32 controls the robot motors and peripherals while the Raspberry Pi captures video frames and transmits them to my laptop.

  • My laptop is in charge of the computer vision related tasks and communicates the relevant information back to the Raspberry Pi using ROS. The Raspberry Pi then executes a state machine and transmits commands to the STM32 over UART.

  • All the details and code (and a demo) are in the GitHub repo above
